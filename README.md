### Hướng dẫn chạy project
1. Mở `cmd`
2. Chạy
```bash
docker compose build
```
3. Chạy
```bash
docker compose up
```
4. Mở link [localhost:8000](http://localhost:8000/)
5. Giao diện web khi chạy thành công
